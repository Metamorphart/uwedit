<b>UWEdit 0.9a</b>

UWEdit is a Sample Editor for Machinedrum UW.
Features:

    Loading and saving Midi Sample Dumps
    Setting Sustain Loop Start/End Points
    Setting Sample Name and Sample Slot
    Sample Upload directly to Machinedrum
